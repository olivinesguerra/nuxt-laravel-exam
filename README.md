

How  to run the API
1. Go to /api
2. Run composer install
3. Run php artisan serve


How  to run the Frontend
1. Go to /ui
2. Run yarn install
3. Run yarn run dev


For the database
1. Install Desktop Docker
2. RUN docker-compose up --build
3. RUN php artisan migrate
